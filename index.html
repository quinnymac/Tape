Welcome to the Tape homepage.

Tape is a open source video player that supports many codecs. Some/most/all of these codecs probably have a different license. I don't have much experience with copyright so I'm putting the creator names along with their links in the about window of the app. I will also have it in the README.

What do we support?

Here's a list. Anything that's not underlined/bold we don't have.

  • MPEG-3 (mp3)
  • MPEG-4 (mp4)

Suggest a issue if you want a codec added.

Current update status:

Not released yet. You can build the source code with Visual Studio 2015 and later. Not guaranteed to work either.
